# PGA Tour Game Scenario Copy

Use this as an editable copy bank for regular season, major, playoff, and FedEx Cup outcomes.

Win rule:
- Always include a tournament win callout when a player wins an event, even if another scenario also applies.

Placeholder notes:
- `<player>`: player name
- `<tournament>`: tournament name
- `<major>`: major name
- `<category>`: strokes gained category
- `<finish>`: finishing position
- `<rank>`: FedEx Cup rank or seed
- `<amount>`: money/prize value
